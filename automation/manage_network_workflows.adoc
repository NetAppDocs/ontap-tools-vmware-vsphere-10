= Manage network access
:icons: font
:imagesdir: ../media/

[.lead]
Use the following API to add IP addresses for whitelisting:

----
patch /api/v1/vcenters/{vcguid}/settings/ip-whitelist
 
{
    value: string
}

GET /api/v1/vcenters/{vcguid}/settings/ip-whitelist
 
{
    value: string
}
----
